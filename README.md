AUTO1 Inventory Performance Project

This folder contains my full analysis and dashboard for the AUTO1 inventory management case study.  
It includes the data preparation, SQL logic, Python exploration and the final Tableau dashboard.

Main Files
- AUTO1_Business_Analyst_Case_Study.ipynb - the main Jupyter notebook with all the steps: data cleaning, calculations and exploratory analysis.  
- Tableau_AUTO1 Inventory Performance Dashboard.twbx - the final Tableau dashboard showing KPIs, sales trends, margins and pricing insights.
- AUTO1_Inventory_Performance_Presentation_Sandeep_Uprety.pptx - PowerPoint presentation summarizing the analysis, key visuals, KPIs and business recommendations.


Folders
- data_sql/ - contains the SQLite database, SQL queries, and the cleaned CSV file used in Tableau.  
  (Files: auto1_inventory.db, auto1_sql_queries.sql, auto1_cleaned_for_sheets.csv)  

- images/ - all the charts and screenshots used in the analysis and summary visuals.  
  (Includes things like chart_days_to_sell.png, chart_margin_by_manufacturer_top10.png, etc.)

  (Includes the PDF and source Excel file.)

How to View
1. Open the Tableau dashboard (.twbx) in Tableau Desktop or Tableau Public to explore the visuals interactively.  
2. Open AUTO1_task.ipynb in Jupyter or VS Code to review the full analysis process.  
3. All key visuals are also saved as .png files in the visuals/ folder for quick reference.
